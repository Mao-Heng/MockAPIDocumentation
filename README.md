# WeatherReport
This endpoint provides the weather forecast for the next hour of a specific city in the US.

### Endpoint
weatherreport/{cityname}

### Parameters
| Parameter name  | Description                         |
| --- | --- |
| {cityname} | The name of the city you want to look up |
